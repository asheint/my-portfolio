SomedayFund

Visit Live Site

A manual expense tracker, budget planner, and savings goal app that needs no bank account linking. Track spending, set budgets, build savings goals, and split trip expenses, all secured with bank-level AES-256 encryption. Free forever, or $9.99 once for Pro, for life.

Built for people who want full control of their money without handing their bank credentials to another app. Your someday starts today.

Why I Built This

Most budgeting apps assume you can link a bank account, but in a lot of countries that kind of bank integration simply isn't available, no matter how many financial apps exist out there. SomedayFund is built around manual tracking instead, low priced but full featured. And the bigger point of the app isn't just logging expenses; it's helping you actually save toward the things you keep telling yourself you'll get to "someday." If you're waiting for the right moment to start saving for a goal, this app is built to set that goal and start funding it today, not someday.

My Contribution

As the founder and full-stack engineer, I designed and built the platform end-to-end across web, mobile, and backend, including:

  • Built the NestJS API

    with Drizzle ORM and PostgreSQL, handling JWT auth with refresh tokens, TOTP-based 2FA, and AES-256-GCM field-level encryption for sensitive financial data.

  • Built the React web app

    covering budgets, envelopes, savings goals, trip expense splitting, analytics, and AI-powered spending insights, with Zustand and TanStack Query handling state and data.

  • Building the mobile app

    using React Native and Expo for iOS and Android, with push notifications, sharing the same NestJS backend as the web app. Currently in development.

  • Integrated billing, monitoring, and deployment

    using Dodo Payments for the one-time lifetime Pro upgrade, Resend for transactional email, and Sentry for error monitoring, with secrets managed via Doppler. The frontend runs on Cloudflare Pages while the API, Postgres, and Redis run on a DigitalOcean droplet.
